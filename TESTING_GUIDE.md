## 📝 Как тестировать отправку сообщений:

### **Способ 1: Использовать два браузера**

**Браузер 1 (Алиса):**
1. Откри `http://localhost:3000`
2. Телефон: `+79991111111`, Имя: `Алиса`
3. Нажми "Войти"

**Браузер 2 (Боб):**
1. Откри `http://localhost:3000` (в другой вкладке или окне)
2. Телефон: `+79992222222`, Имя: `Боб`
3. Нажми "Войти"

**Отправь сообщение Алиса → Боб:**
1. В левой панели введи телефон: `+79992222222`
2. Нажми "Новый чат"
3. Должен появиться чат с Бобом
4. Напиши сообщение и отправь
5. Открой окно с Бобом - сообщение должно появиться!

### **Способ 2: Использовать консоль браузера**

Если у тебя только один браузер, протестируй через консоль:

1. Откри браузер: `http://localhost:3000`
2. Нажми F12 (открыть DevTools)
3. Перейди на вкладку **Console**
4. Выполни:

```javascript
// Проверить, что socket подключился
console.log("Socket ID:", socket.id);

// Проверить userId
console.log("User ID:", userId);

// Отправить тестовое сообщение
socket.emit("send_message", {
  sender_id: 1,
  receiver_id: 2,
  text: "Тестовое сообщение от консоли"
});

// Проверить в БД
fetch('/api/messages/dialog/1/2').then(r => r.json()).then(d => console.log(d));
```

## ❌ Если что-то не работает:

### **1. Сервер не запускается**
```bash
# Проверь логи
node /root/messenger/server.js 2>&1 | head -20

# Проверь порт
lsof -i :3000

# Убирай занятой порт
pkill -9 node
```

### **2. Фронт не подключается к серверу**
- Откри DevTools (F12)
- Вкладка Network/WebSocket
- Должно быть подключение к `ws://localhost:3000/socket.io/...`

Если нет - проверь консоль на ошибки.

### **3. Сообщение не сохраняется**
Проверь в консоли что не пусто:
- `sender_id` 
- `receiver_id`
- `text`

И проверь логи сервера на ошибки.

### **4. Чаты не создаются**
```bash
# Проверь в БД
sudo -u postgres psql -d messenger -c "SELECT * FROM chat_users;"

# Должны быть строки вроде:
# chat_id | user_id
#    1    |   1
#    1    |   2
```
