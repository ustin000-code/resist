## 🐛 Если что-то не работает:

1. Проверь логи сервера (должны быть эмодзи и подробная информация)
2. Убедись, что пользователи существуют в БД:
   ```bash
   sudo -u postgres psql -d messenger -c "SELECT id, name FROM users;" 
   ```
3. Проверь таблицу messages:
   ```bash
   sudo -u postgres psql -d messenger -c "SELECT * FROM messages LIMIT 5;"
   ```
