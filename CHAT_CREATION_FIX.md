## 🚀 КАК ТЕСТИРОВАТЬ:

### 1️⃣ **Запусти сервер:**
```bash
cd /root/messenger
node server.js
```

### 2️⃣ **Открой браузер:**
```
http://localhost:3000
```

### 3️⃣ **Вход (левая панель):**
- Телефон: `+79991111111`
- Имя: `Алиса`
- Нажми "Войти"

### 4️⃣ **Создать чат (два способа):**

**Способ 1: Выбрать из списка контактов**
- Видишь список "Контакты:" с именами и телефонами
- Клики на контакт → автоматически создается чат

**Способ 2: Ввести телефон вручную**
- Введи телефон: `+79992222222`
- Нажми "Создать"

### 5️⃣ **Отправить сообщение:**
- Напиши текст в нижнем поле
- Нажми Enter или кнопку "Отправить"

## 📋 ЛОГИКА РАБОТЫ:

```
1. Пользователь жмёт на контакт или вводит телефон
   ↓
2. Фронт отправляет POST на /api/chats/create
   ↓
3. Бэкенд:
   - Берет текущего пользователя из токена
   - Ищет другого пользователя по телефону
   - Проверяет, есть ли уже чат между ними
   - Если нет → создает новый чат
   ↓
4. Возвращает chat_id и other_user_id
   ↓
5. Фронт открывает чат и показывает историю
```
