# 🎯 БЫСТРАЯ ИНСТРУКЦИЯ

## Как протестировать чаты во фронте:

### 1️⃣ Запусти сервер в одном терминале:
```bash
cd /root/messenger
node server.js
```

Должно вывести:
```
✅ Сервер запущен на http://localhost:3000
📊 WebSocket сервер готов к подключениям
```

### 2️⃣ Открой браузер:
```
http://localhost:3000
```

### 3️⃣ Зарегистрируйся (левая панель):
- Телефон: `+79991111111`
- Имя: `Алиса`
- Нажми "Войти"

### 4️⃣ Создай новый чат:
- В левой панели введи: `+79992222222`
- Нажми "Новый чат"
- Должен появиться чат

### 5️⃣ Отправь сообщение:
- Напиши в текстовом поле: "Привет!"
- Нажми "Отправить" или Enter
- Сообщение должно появиться в окне

---

## ⚠️ Если не работает:

1. **Сервер не запускается** → `pkill -9 node` → запусти заново
2. **Нет сообщений** → посмотри логи сервера (ошибки)
3. **Фронт не подключается** → проверь DevTools F12 → Network → должно быть подключение WebSocket
