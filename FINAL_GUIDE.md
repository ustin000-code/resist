# ✅ ФИНАЛЬНАЯ ИНСТРУКЦИЯ ПО ТЕСТИРОВАНИЮ

## 🚀 ЗАПУСК:

### **Шаг 1: Запусти сервер**
```bash
cd /root/messenger
node server.js
```

Должно вывести:
```
✅ Сервер запущен на http://localhost:3000
📊 WebSocket сервер готов к подключениям
✅ БД подключена
```

### **Шаг 2: Открой браузер**
```
http://localhost:3000
```

---

## 🧪 ТЕСТИРОВАНИЕ СОЗДАНИЯ ЧАТА:

### **Способ 1: Вход и клик на контакт**

1. **Вход в систему:**
   - Телефон: `+79991111111`
   - Имя: `Алиса`
   - Нажми "Войти"

2. **Открой DevTools:**
   - Нажми F12
   - Перейди на вкладку Console
   - Там должны быть зелёные логи вроде:
     ```
     ✅ Восстановлен пользователь: Алиса
     👥 Контакты загружены: [...]
     ✅ Контакты отображены
     ```

3. **Клик на контакт:**
   - Видишь список в левой панели "Контакты:"
   - Клики на "Боб"

4. **Проверь консоль:**
   - Должны быть логи:
     ```
     🖱️ Клик на контакт: Боб +79992222222
     📞 Создание чата с номером: +79992222222
     ✅ NEW CHAT: {chat_id: 1, other_user_id: 2}
     ```

5. **Чат должен отобразиться:**
   - В списке чатов (левая панель)
   - Откроется автоматически

---

### **Способ 2: Ввод телефона вручную**

1. **После входа:**
   - В левой панели найди "Новый чат"
   - Введи телефон: `+79992222222`
   - Нажми "Создать"

2. **Проверь консоль на логи**

3. **Чат должен создаться**

## 🔥 ПОИСК ОШИБОК:

### **Нет контактов в списке:**
```javascript
// Выполни в console (F12)
console.log("Контакты:", document.getElementById("contactsList").innerHTML);
```
Если пусто - проверь логи выше для ошибок загрузки.

### **Клик не работает:**
```javascript
// Выполни в console
document.getElementById("contactsList").querySelectorAll("li")[0].click();
```
Если работает вручную - проблема в обработчике события.

### **Чат создается но не отображается:**
```javascript
// Проверь в БД чаты
// В отдельном терминале:
sudo -u postgres psql -d messenger -c "SELECT * FROM chat_users WHERE user_id = 1;"
```

## ✨ ЕСЛИ ВСЕ РАБОТАЕТ:

После клика на контакт должно быть:
- ✅ Зелёные логи в консоли браузера
- ✅ Новый чат в списке чатов (левая панель)
- ✅ Автоматическое открытие чата
- ✅ История пустая (если это новый чат)
- ✅ Можно писать сообщения

## 🆘 ЕСЛИ НЕ РАБОТАЕТ:

1. **Очисти localStorage и перезагрузись:**
   ```javascript
   localStorage.clear();
   location.reload();
   ```

2. **Добавь тестовых пользователей:**
   ```bash
   node add_test_users.js
   ```

3. **Проверь БД:**
   ```bash
   sudo -u postgres psql -d messenger -c "SELECT id, name, phone FROM users;"
   ```

4. **Перезапусти сервер:**
   ```bash
   pkill -9 node
   node server.js
   ```

5. **Посмотри полные логи:**
   - Консоль браузера (F12)
   - Консоль сервера (Node.js)
